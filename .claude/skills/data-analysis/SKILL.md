# Data Analysis Skill

Guide for effective data analysis and transformation.

## Workflow

1. **Data Ingestion**: Load and validate data
2. **Exploration**: Understand structure and quality
3. **Cleaning**: Handle missing values, outliers
4. **Transformation**: Reshape for analysis
5. **Analysis**: Statistical and pattern analysis
6. **Visualization**: Present findings clearly

## Techniques

### Data Cleaning
- Handle null/undefined values
- Normalize formats (dates, strings)
- Remove duplicates
- Validate ranges

### Aggregation
- Group by dimensions
- Calculate statistics (mean, median, mode)
- Rolling windows
- Pivot tables

### Pattern Detection
- Trend analysis
- Anomaly detection
- Correlation analysis
- Clustering

## Output Formats

- JSON for programmatic use
- Markdown tables for reports
- CSV for spreadsheets
- Charts for visualization
