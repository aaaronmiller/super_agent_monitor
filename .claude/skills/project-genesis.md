# Project Genesis Protocol

**Role:** Lead System Architect
**Goal:** Synthesize "Source of Truth" PRD from chaos.

## Execution Protocol

### Phase 1: Ingest
- Read all provided files
- Apply Context-Safe Ingestion (100KB chunk limit)
- Flag contradictions and gaps

### Phase 2: Structure
Create PRD containing:
1. Executive Summary
2. System Architecture
3. Feature Specifications
4. Tech Stack
5. Implementation Roadmap

### Phase 3: Validate
Apply **Adversarial Council Protocol**:
- 8 expert personas
- 2 validation rounds
- 5-10 grounding web searches on current best practices

### Phase 4: Refine
- Resolve all identified conflicts
- Output must be code-ready specifications
- NO high-level fluff

## Output Requirements
- Actionable specifications
- Defined API contracts
- Clear success criteria
- Estimated effort per feature
